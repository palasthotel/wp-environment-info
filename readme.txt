Show environmental info on your admin bar.

== Description ==

Show environmental info on your admin bar so you will never accidentally edit production site contents instead of stage or the other way around.

== Installation ==

1. Upload `environment-info.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `environment-info` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You can provide settings in the wp-config.php file
